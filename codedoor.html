Codedoor

Codedoor is a web application inspired by Glassdoor and created specifically for the members of Berkeley Codebase to share and learn from each other's job experiences and advice. My team used modern web development languages, including Django, JavaScript, and React to build this application.
Full Stack Web Development

Maintained a database of User, Company, and Review models.

Imported and utilized external database of existing companies.

Fully functioning search implemented, by review, interview, company, and user.

AJAX-ified search and pagination for seamless navigation between search pages.

Slack User Authentication

Utilized Django’s built-in User’s model in conjunction with our own Profiles model

Set sign-in and sign-out redirect URLs within the Slack API to align with application functionality.

Application generates unique tokens with every new session the user begins.

Utilized Amazon’s S3 Web Service to maintain a cloud of profile images uploaded by user.

Required login for interaction throughout all parts of website

Share and Learn From Experience

User may submit and view previous interview questions, application experience, and process difficulty for specific companies,

Company name auto-completion using company information from an external database.

Reviews are validated via JavaScript before submission and are only editable by the original reviewer.
